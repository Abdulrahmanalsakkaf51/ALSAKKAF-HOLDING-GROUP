# 1. Purpose

This template is used before a Partner uses paid AI, premium AI, ads, subscriptions, or paid tools beyond already-approved limits, consistent with the CEO Approval Gates in APFA-001 Section 17 and the budget rules in ADDA-001.

No Partner may incur spend beyond its approved Cost Rules without a completed and approved version of this template.

---

# 2. How to Use This Template

1. Copy this file whenever a Partner (or Atlas on a Partner's behalf) identifies a need for spend beyond current approval.
2. Fill in every field with a realistic, non-inflated estimate.
3. Submit to the Founder before any spend occurs.
4. File the completed, decided template with the Partner's cost records.
