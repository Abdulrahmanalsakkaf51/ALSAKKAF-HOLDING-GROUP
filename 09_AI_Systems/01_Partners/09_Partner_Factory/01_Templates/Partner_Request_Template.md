# 1. Purpose

This template is used when the Founder, Atlas, or a department proposes a new Partner.

It captures the request stage of the Partner Creation Lifecycle (PCL-001, Section 3.2), before any profile, skill, or prompt work begins.

---

# 2. How to Use This Template

1. Copy this file into a working note or the requesting department's records.
2. Fill in every field below.
3. Submit the completed request to the Founder (directly, or through Atlas) for review before any Partner Profile is drafted.
4. If accepted, the request becomes the input to `Partner_Profile_Template.md`.
