# 1. Purpose

This template is used to review an active Partner's performance over a defined period, following the Partner Performance Monitoring Model in APFA-001 Section 13.

---

# 2. How to Use This Template

1. Copy this file for each review cycle and store it alongside the Partner's records.
2. Complete the review using real task, cost, and outcome data — do not invent figures.
3. Route improvement proposals through `Partner_Learning_Log_Template.md`.
4. Attach the final decision to the Partner Registry entry.
