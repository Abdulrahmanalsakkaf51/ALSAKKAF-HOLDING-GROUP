# 1. Purpose

This template is used to record learning from Partner work, following the Partner Learning System (PLS-001) and the Learning Loop discipline defined for AOS.

Learning entries feed approved skill or prompt updates — they never trigger silent, unapproved self-modification.

---

# 2. How to Use This Template

1. Copy this file each time a Partner task or event produces a notable lesson.
2. Fill in every field honestly, including failures.
3. If the entry proposes a skill or prompt update, route it through Founder approval before any change is made.
4. Note whether The Librarian should index this entry for future discoverability.
