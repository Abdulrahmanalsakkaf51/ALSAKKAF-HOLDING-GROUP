# 1. Purpose

This template is used when Atlas assigns work to a Partner, following the Partner Task Assignment Model in APFA-001 Section 12.

---

# 2. How to Use This Template

1. Copy this file for each task assignment, or log entries in a shared task assignment record.
2. Fill in every field before the Partner begins work.
3. Update progress status and completion result as the task moves forward.
4. File the completed record where the requesting authority can review it.
