# 1. Purpose

This template is used to test a Partner or a specific Partner skill before activation, following the Test Log stage of the Partner Creation Lifecycle (PCL-001 Section 3.9).

---

# 2. How to Use This Template

1. Copy this file into `09_AI_Systems/01_Partners/03_Test_Logs/` and rename it `PARTNER-0XX_[Name]_Test_Log.md`.
2. Run realistic test cases against the Partner or skill before it is proposed for activation.
3. Record honest results, including failures.
4. Attach this test log to the Partner Profile before requesting Founder approval.
