# 1. Purpose

The Librarian Document Map identifies the main documents The Librarian should know, search, reference, and explain.

This map helps The Librarian answer questions such as:

- Where is AOS defined?
- Which document explains The Mind?
- Which ADR approved the Partner Registry?
- Where is the 30-Partner Company Cell explained?
- Which file should be updated when a new Partner is created?
- Which records are approved, designed, draft, or proposed?

This document helps The Librarian become more reliable.
