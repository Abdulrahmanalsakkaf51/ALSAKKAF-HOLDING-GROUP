# 1. Before Any Document Is Considered Done

| # | Check |
|---|-------|
| 1 | Correct template used (report, minutes, letter, tracker) |
| 2 | Title, date, and reporting period present and correct |
| 3 | All numbers computed from real data — never invented |
| 4 | Names, titles, and organizations spelled correctly |
| 5 | No placeholder tokens ({{...}}) left in the final file |
| 6 | Tables have headers; no empty dangling rows |
| 7 | File named per the Office File Management Standard (OFS-001) |
| 8 | "Prepared by / Reviewed by" note present on drafts |
| 9 | No credentials, passwords, or private data in the document |
| 10 | Human reviewer identified before the document is shared |

# 2. Spreadsheet-Specific Checks

# 3. Email-Draft-Specific Checks

| # | Check |
|---|-------|

| 2 | Recipient, subject, and topic correct |
