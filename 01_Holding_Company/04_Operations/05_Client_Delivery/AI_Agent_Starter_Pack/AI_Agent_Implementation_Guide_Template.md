# 3. Setup (15 minutes, one time)

1. Open your AI tool ([client's tool from intake, e.g., Claude or ChatGPT]).
2. Create three separate conversations or projects — one per agent. Name them: "[Manager Agent name]", "[Research Agent name]", "[Operations Agent name]".
3. Paste each agent's prompt as the first message (or into the custom instructions / project instructions slot if your tool has one).
4. Send each agent a test greeting. Each should introduce itself and ask what to work on.
5. Keep the routing map open or printed next to you for the first week.

---

# 4. Your First Week

| Day | Do this |
|-----|---------|
| 1 | Give the Manager Agent your real to-do list; ask it to prioritize your day |
| 2 | Ask the Research Agent one real question about a competitor or customers |
| 3 | Ask the Operations Agent to turn one recent decision into a checklist |
| 4 | Try one handoff chain from the routing map |
| 5 | Ask the Manager Agent for a week summary and next week's top 3 priorities |

Small, real tasks beat big experiments. Use your actual business situations from day one.

---

# 5. Your 30-Minute Setup Session

During the included session we will: load the three prompts together, run one real task per agent, walk through the routing map, and answer your questions. Booking: [date/time to be arranged at handover].

---

# 6. Getting Better Results

- Give context: the more real detail you provide, the more specific the output.
- Correct the agent when output misses — it adjusts within the conversation.
- Re-paste the prompt in a fresh conversation whenever an agent drifts from its role.
- Revisit the profiles after a month; if your business changed, the revision rules ([AIAD-009 reference]) explain how updates work.
