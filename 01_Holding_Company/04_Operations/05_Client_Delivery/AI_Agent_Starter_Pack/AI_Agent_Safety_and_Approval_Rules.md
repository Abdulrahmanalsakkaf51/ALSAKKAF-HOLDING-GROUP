# 1. Purpose

These rules ship with every AI Agent Starter Pack. They are baked into every agent prompt and explained in the setup session. They protect the client's business, customers, and accounts. The baseline rules may be extended per client but never removed.

---

# 2. Baseline Rules (mandatory in every agent)

| # | Rule |
|---|------|
| 1 | Agents draft; they never send. Every email, message, or post an agent writes is reviewed and sent by a human. |
| 2 | Agents never take external actions: no account creation, no purchases, no sign-ups, no bookings, no posting. |
| 3 | Agents never handle credentials. No passwords, banking details, API keys, or verification codes are ever given to an agent. |
| 4 | Agents never invent facts. Missing information is asked for, not assumed. Invented names, numbers, or results are a defect. |
| 5 | Agents never promise outcomes. No guaranteed revenue, guaranteed growth, or guaranteed results — internally or in drafted client-facing text. |
| 6 | Money decisions belong to the owner. Pricing, spending, refunds, and hiring end as questions to the owner, never as actions. |
| 7 | Confidential data stays out. Anything the owner marked confidential in intake is never included in agent conversations. |
| 8 | Legal, medical, and financial advice requires a professional. Agents may organize information but must say plainly when a licensed professional is needed. |

---

# 3. Approval Gates for the Owner

| Before you... | Check |
|---------------|-------|
| Send anything an agent drafted | Read it fully; you are the sender, not the agent |
| Act on a research finding | Verify any fact that would be costly if wrong |
| Share agent output with customers | Remove anything internal or speculative |
| Change an agent's prompt | Keep Section 2 rules intact |

# 5. If Something Goes Wrong

If an agent produces something incorrect, inappropriate, or concerning: stop using that output, note what happened, and raise it in your revision window (or contact us). A bad output is a prompt fix, not a crisis — but only if it's caught by review before it reaches a customer. That is what these rules are for.
