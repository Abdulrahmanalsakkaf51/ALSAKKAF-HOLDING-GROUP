# 1. Project Purpose

Build the operating structure for ALSAKKAF HOLDING GROUP media operations (three YouTube channels), the Content Factory production line that Atlas and Partners will operate, and the client-facing demonstration and education assets for ALSAKKAF Systems — so media execution, customer understanding, and honest reporting all run on documented rails.

# 4. Standing Rules

1. Nothing is published, uploaded, or scheduled without explicit Founder approval — the calendars are plans, not authority.
2. Original or licensed content only; rights confirmed before an item leaves Idea status.
3. Analytics are computed from real tracker rows; zero stays zero; no estimates.
4. No YouTube credentials or API keys anywhere in the repository.
5. Client-facing assets stay consistent with CDP-001; fictional demos are labeled fictional.
