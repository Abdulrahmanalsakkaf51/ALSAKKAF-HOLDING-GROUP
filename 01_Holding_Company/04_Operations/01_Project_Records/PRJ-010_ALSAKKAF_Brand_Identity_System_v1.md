# 1. Objective

Create the first production brand identity system for the three levels of the brand:

1. ALSAKKAF HOLDING GROUP — master brand.
2. ALSAKKAF Systems — public operating company.
3. AOS AI Services — service line.

All three share one original geometric "A" foundation, built as clean original SVG geometry — not raster AI-generated logo text.

---

# 2. Approved Design Direction

| Element | Direction |
|---------|-----------|
| Master direction | White background master direction |
| Style | Modern corporate, premium, futuristic but trustworthy |
| Foundation | Geometric "A" symbol |
| Core colors | Navy, Electric Blue, Cyan |
| Service accent | Optional Violet for AOS AI Services |

The A represents: Ambition, Architecture, Artificial Intelligence, Advancement, ALSAKKAF.

# 4. Out of Scope

- Trademark registration.
- Print production files (CMYK).
- Paid font licensing. Wordmark text uses a system font stack in v1; converting wordmarks to vector outlines is a v2 task.
- Raster AI-generated logo art.

---

# 5. Success Criteria

1. All nine logo SVG files exist and render.
2. Favicon and mobile icons load on the live site.
3. Open Graph metadata references the brand OG image.
4. All PNG assets are generated by a repeatable local tool using only the Python standard library.
5. Markdown Audit passes for brand documentation.
