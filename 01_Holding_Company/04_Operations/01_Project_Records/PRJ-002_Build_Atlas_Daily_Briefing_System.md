# 1. Purpose

PRJ-002 exists to turn Atlas Daily Briefing from an approved workflow into a practical working Founder support system.

Atlas is already active as PARTNER-002 — Founder Executive Partner.

The first active Atlas feature is Atlas Daily Briefing.

This project will define, test, and improve how Atlas gives the Founder a daily briefing using known company context, active project records, Partner status, risks, missing information, and recommended next actions.

---

# 2. Project Goal

The goal of PRJ-002 is to create a reliable daily briefing system that helps the Founder answer:

```text
Atlas, what should I focus on today?
```

The briefing should help the Founder see:

| Briefing Item | Purpose |
|--------------|---------|
| Current Focus | Identify the most important focus for the day |
| Active Projects | Show which projects currently need work |
| Partner Status | Show which Partners are active or relevant |
| Pending Decisions | Show what needs Founder approval |
| Follow-Ups | Show known follow-ups or missing follow-up information |
| Risks | Show risks the Founder should know |
| Missing Information | Show what Atlas does not yet know or cannot access |
| Recommended Priorities | Give practical priorities in order |
| Next Action | Give one clear next step |

# 6. Authority Rules

Atlas may:

| Allowed Action | Meaning |
|---------------|---------|
| Assist | Help the Founder understand information |
| Summarize | Summarize projects, Partners, and records |
| Organize | Structure priorities, tasks, and next steps |
| Recommend | Suggest options and priorities |
| Prepare Briefings | Prepare Founder briefings for review |

Atlas may not:

| Restricted Action | Reason |
|-------------------|--------|
| Approve decisions | Final authority remains with the Founder |
| Send messages | Requires Founder approval |
| Spend money | Requires Founder approval |
| Edit official records without approval | Protects company records |
| Access private systems without approved rules | Protects data and security |
| Control the computer | Requires future safety controls |
| Replace the Founder | Atlas supports the Founder only |
