# Purpose

Give Atlas short, reusable, honest phrase blocks for the common reply situations, so drafts stay consistent with the live website and never drift into guaranteed-results language, fake urgency, or invented social proof.

---

# Scope

Covers phrase blocks for acknowledgement, discovery call invitation, demo invitation, pricing explanation, scope clarification, and polite decline. Every phrase block is a starting point to be adapted to the actual message — never copy-pasted without reading the original message first.

---

# Rules That Apply to Every Phrase Block

- No guaranteed-results language ("this will double your revenue," "guaranteed ROI," etc.).
- No fake urgency ("only 2 spots left today," "offer ends tonight" — unless a real, Founder-confirmed, time-boxed offer like the Founding Client Pilot actually exists and is stated exactly as documented).
- No invented social proof — no client names, counts, or testimonials that are not already Founder-approved and published.
- Pricing language matches `docs/index.html` (the live site) verbatim on figures and offer names — see Section 3 below.

---

# 1. Acknowledgement

```text
Thanks for reaching out — got your message and will get back to you
with a proper answer shortly.
```

```text
Appreciate you taking the time to write in. Let me look into this
and follow up with specifics.
```

---

# 2. Discovery Call Invitation

```text
Happy to jump on a short call if that's easier — no pressure, just a
conversation to see if this is a good fit for what you're working on.
```

```text
If it's useful, I'm glad to do a quick 15-20 minute call to talk
through what you need before anything else.
```

---

# 3. Demo Invitation

```text
If it helps, you can see how the AI Partner system works in a safe,
scripted demonstration mode at [Partner Playground link] — no live
data involved, just a walkthrough of how it operates.
```

```text
Happy to walk you through a short demo of how the workflow/agent
system works before you decide anything.
```

Note: the Partner Playground (`docs/partner-playground.html`) is a safe demonstration mode only — do not describe it as a live, connected system.

---

# 4. Pricing Explanation

Use these figures and offer names exactly as they appear on the live site (`docs/index.html`) — do not round, discount, or rephrase the numbers.

```text
We currently have two fixed-scope packages:

- The AOS AI Workflow Starter Pack — $399 USD, 3 business
  automations for the price of one (lead/client tracking, one
  internal operations workflow, and a CEO/management visibility
  view), delivered in 5-7 business days.
- The AOS AI Agent Starter Pack — $450 USD, 3 AI agents for the
  price of one (Manager, Research/Analyst, and Operations agent
  roles, with prompts, task routing, and a guided setup session),
  delivered in 5-7 business days.

Both are one-time payments for a one-time delivery, not
subscriptions. If what you need falls outside either package,
that's a Request Custom Quote conversation — I'll follow up before
any payment is involved.
```

```text
Short version: $399 for the Workflow Starter Pack, $450 for the
Agent Starter Pack. Anything custom, we talk first — Request Custom
Quote, no payment involved until it's scoped and agreed.
```

The PayPal payment links themselves ($399 pack: `https://www.paypal.com/ncp/payment/2AN8FH99X682C`; $450 pack: `https://www.paypal.com/ncp/payment/2WXPECSR3UH68`) follow the hard rule in `Reply_Draft_Template.md` (COMMS-003): never included in a first cold reply, only after the lead has shown confirmed interest.

# 5. Scope Clarification

```text
Just to make sure I answer the right question — could you tell me a
bit more about [specific thing they mentioned]? That'll help me give
you an accurate answer instead of a generic one.
```

```text
To be clear on what's included: the Starter Pack is a single scoped
engagement (see the deliverables list on the site) — it's not an
open-ended or unlimited-scope offer. If you need something broader,
that's a custom conversation.
```

---

# 6. Polite Decline

```text
Thanks for thinking of us, but this isn't something we're able to
take on right now — wishing you well with it.
```

```text
Appreciate the interest, but this falls outside what we currently
offer. If that changes, feel free to reach back out.
```

# How to Use

1. Adapt every phrase block to the actual message received — never send a phrase block unedited without reading the original message first.
2. If a reply needs to say something not covered here, draft it fresh using the Reply Draft Template (COMMS-003) and the same rules (no guarantees, no fake urgency, no invented proof) rather than stretching a phrase block to fit.
3. If the topic matches the Escalation Matrix (COMMS-007), use that document's "Safe Holding Reply" wording instead of the Pricing/Scope blocks here.
4. Keep this library in sync with `docs/index.html` — if the live site's pricing or offer wording changes, update this file in the same session.
