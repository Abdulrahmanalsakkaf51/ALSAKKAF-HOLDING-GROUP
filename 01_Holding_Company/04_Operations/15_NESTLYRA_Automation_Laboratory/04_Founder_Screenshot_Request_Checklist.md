# 2. Capture Rules

1. Capture the full settings page, including any warning banners — banners are evidence.
2. If a page has multiple tabs or expandable sections (for example each shipping zone), capture each expanded state.
3. Do not change any setting while capturing. If Shopify prompts for a change, dismiss it.
4. Date the screenshots (filename or note) so each fact in the Current-State Audit carries an evidence date.
5. Store screenshots outside this public repository, in the Founder's private storage, per PODS-001. Only the verified facts (settings, states) are transcribed into NAL-002 — never the raw images, which may contain account details.
