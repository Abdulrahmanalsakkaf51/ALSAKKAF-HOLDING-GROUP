## Purpose

The shape of a batch of outreach drafts Atlas's `atlas.py outreach` command generates for CEO review. **Nothing in this batch is sent by Atlas.**

---

## Template

**Batch date:** [Date]
**Channel:** [Email / LinkedIn / Instagram / WhatsApp]

### Draft 1
- Recipient: [Lead ID / Company Name placeholder]
- Personalization note: [Specific observed detail]
- Message body: [Drafted message]
- CEO Approved: [ ] Yes  [ ] No

### Draft 2
- Recipient: [Lead ID / Company Name placeholder]
- Personalization note: [Specific observed detail]
- Message body: [Drafted message]
- CEO Approved: [ ] Yes  [ ] No

*(Repeat per lead in the batch.)*

---

## How to Use

1. Every draft requires an explicit CEO Approved: Yes before it may be sent.
2. Sending itself is always a manual, CEO-controlled action — never automated.
3. Do not include the PayPal payment link in a first cold draft.
