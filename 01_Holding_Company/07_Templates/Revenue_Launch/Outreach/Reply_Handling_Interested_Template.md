# Purpose

The reply to send when a prospect responds positively to a first-contact message (e.g., "sounds interesting," "tell me more," "yes let's talk").

---

# Template

```text
Hi [First Name] — great to hear, thank you.

The fastest next step is usually a short 15-minute call so I can ask a few questions about how [Company Name] currently handles [specific workflow named in the original outreach], then confirm the exact scope before anything is quoted or built.

Would [propose 2 specific time slots] work, or would you rather I just send a short written outline first?

Best,
[Your Name]
```

---

# How to Use

1. CEO approval is required before this message is sent.
2. Do not include the PayPal payment link in this reply — confirm scope first, on a call or in writing.
3. Reference the specific workflow problem already named in the original outreach message; do not introduce a new claim.
4. If the prospect skips the call and asks directly for price or a demo, use RLOUT-010 (Asks for Price) or RLOUT-011 (Asks for Demo) instead.
5. Log the reply and this response in the Communications Inbox Register (`01_Holding_Company/04_Operations/13_Client_Communications/`).
