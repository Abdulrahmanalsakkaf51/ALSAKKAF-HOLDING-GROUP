# Purpose


# How to Use

1. CEO approval is required before this message is sent.
2. Send only if there has been no reply to Follow-Up 1, per the Follow-Up Schedule in STRAT-008.
3. This is the first template where mentioning the $399 price is acceptable — as a closing-the-loop reference, not a hard sell. The PayPal link itself is still only sent once the client explicitly confirms interest.
4. This is the last scheduled follow-up — do not continue messaging after this without a reply.
