# How to Use

1. Add one row per lead as soon as it is identified.
2. Update Status as the lead moves through the pipeline.
3. "Contacted" only applies after the CEO has approved and sent the outreach — see STRAT-008.
4. Never store passwords, recovery codes, API keys, or payment card data in this file.
