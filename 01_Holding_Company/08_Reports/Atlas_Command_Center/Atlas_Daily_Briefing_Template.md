# 1. Purpose

This is the revenue-launch-specific daily briefing Atlas fills during PRJ-007. It is not a replacement for the general Atlas Daily Briefing Workflow (`ABRIEF-001`) — it is a focused variant covering only revenue-launch activity: leads, outreach, content, and cash.

---

# 2. Template

```text
Date: [YYYY-MM-DD]

Revenue Collected So Far: $[amount]

Leads Found Today: [number]
Leads Qualified Today: [number]

Outreach Drafted Today: [number]
Outreach Sent Today (CEO-approved only): [number]
Replies Received Today: [number]

Content Published Today: [number]

Pending CEO Decisions:
- [decision 1]
- [decision 2]

Blockers:
- [blocker 1]

Recommended Focus for Tomorrow:
[one clear recommendation]
```

---

# 3. Example — Day 0 (Launch Day, Before Any Activity)

```text
Date: 2026-07-13

Revenue Collected So Far: $0

Leads Found Today: 0
Leads Qualified Today: 0

Outreach Drafted Today: 0
Outreach Sent Today (CEO-approved only): 0
Replies Received Today: 0

Content Published Today: 0

Pending CEO Decisions:
- Choose contact email
- Approve landing page copy
- Approve first outreach batch

Blockers:
- No contact email chosen yet
- Website not yet published
- No real leads loaded yet

Recommended Focus for Tomorrow:
Choose the contact email and approve the landing page — every other step depends on these two.
```

---

# 4. How to Use

1. Atlas fills this template once per active launch day, using data from `03_Revenue_Operations/` CSVs, `04_Content_Operations/` CSVs, and the Atlas Revenue Dashboard (ACC-003).
2. Never mark "Outreach Sent Today" above 0 unless the CEO explicitly approved that specific batch and it was actually sent through an approved method.
3. Numbers must be honest — if nothing happened, the briefing says 0, not an estimate.
