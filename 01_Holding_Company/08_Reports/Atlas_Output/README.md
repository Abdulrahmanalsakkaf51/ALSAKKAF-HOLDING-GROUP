# 1. What This Folder Is

This is the working output folder for the Atlas Runtime local tool (`09_AI_Systems/02_Tools/Atlas_Runtime/atlas.py`). Every file in this folder (and its subfolders) is generated locally by running an Atlas Runtime command — nothing here is written by hand as a source document, and nothing here is sent, published, or acted on automatically.

---

# 2. What Atlas Generates Here

| Subfolder | Generated By | Contents |
|-----------|--------------|----------|
| (root) | `atlas.py brief`, `atlas.py war-room` | Daily CEO briefings, the War Room report |
| `Proposal_Drafts/` | `atlas.py proposal` | Draft client proposals for the AI Workflow Starter Pack |
| `Outreach_Drafts/` | `atlas.py outreach` | Draft outreach message batches (never sent automatically) |
| `Content_Packs/` | `atlas.py content-pack` | Daily multi-platform content drafts |
| `Client_Delivery/` | `atlas.py delivery-checklist` | Per-client delivery checklists |
| `Revenue_Reports/` | `atlas.py payment-report`, `atlas.py war-room` | Payment/pipeline status reports |

Lead scoring reports (`atlas.py score-leads`) are also written to the root of this folder.

---

# 3. What Is Safe to Store Here

- Draft proposals, outreach messages, content, checklists, and reports
- Payment status pulled from `Client_Pipeline.csv` (price, link-sent status, paid/unpaid status)
- The public PayPal payment link (`https://www.paypal.com/ncp/payment/2AN8FH99X682C`)
- Company names, service interest, and other business-context fields already present in the trackers

---

# 4. What Must Never Be Stored Here

Never — in any file Atlas generates or the Founder adds to this folder:

- PayPal login, password, or 2FA codes
- Recovery codes or account recovery data
- API keys or secret keys
- Bank account or card details

If a task ever seems to require any of the above, stop and handle it outside this repository, consistent with Guardian's credential-handling rule (`PARTNER-016_Guardian.md`) and `AOS_Payment_Portal_Readiness_Plan.md` (STRAT-014).

# 5. How to Run Atlas Runtime

From the repository root:

```text
py "09_AI_Systems\02_Tools\Atlas_Runtime\atlas.py" health-check
py "09_AI_Systems\02_Tools\Atlas_Runtime\atlas.py" brief
py "09_AI_Systems\02_Tools\Atlas_Runtime\atlas.py" war-room
```

Or double-click `RUN_ATLAS_COMMAND_CENTER.bat` (or run `RUN_ATLAS_COMMAND_CENTER.ps1`) at the repository root to run the War Room report directly.

Every command is local, read/write-only inside this repository, and requires no network access, no API key, and no login of any kind.
