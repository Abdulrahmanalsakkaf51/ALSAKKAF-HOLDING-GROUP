# Decision

ALSAKKAF HOLDING GROUP shall use a Project Operating Model to define how projects are created, approved, managed, tracked, completed, and converted into institutional knowledge.

---

# Reason

The institution is now moving from foundation documentation into execution.

Without a project model, ideas may become random tasks without clear owners, scope, risks, deliverables, or learning.

The Project Operating Model ensures that work becomes structured, traceable, and connected to AOS.

---

# Consequences

Official projects must have a project ID, owner, purpose, scope, deliverables, success criteria, risks, status, and project record.

Partners may support projects, but human accountability remains required.

The first recommended project is PRJ-001 — Build The Librarian Tool.
