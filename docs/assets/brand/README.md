# 1. Purpose

This folder holds the production brand assets for the three levels of the brand:

1. ALSAKKAF HOLDING GROUP — master brand (long-term parent vision).
2. ALSAKKAF Systems — public operating company.
3. AOS AI Services — service line.

All assets are original SVG geometry built inside this repository. No raster AI-generated logo text is used as a production logo.

---

# 2. The Mark

The geometric A is built from three planes:

| Plane | Color | Meaning |
|-------|-------|---------|
| Left leg | Navy | Architecture, stability, trust |
| Right leg | Electric Blue to Cyan gradient | Advancement, technology |
| Floating crossbar | Cyan (Violet for AOS AI Services) | Artificial Intelligence, the connecting layer |

The A represents: Ambition, Architecture, Artificial Intelligence, Advancement, ALSAKKAF.

Family variants:

- Holding Group: the pure two-tone A.
- Systems: the A with network nodes at its three vertices.
- AOS AI Services: the A with a violet AI orbit.

# 5. Regenerating Raster Assets

All PNG files are generated by a repeatable local tool using only the Python standard library:

`py 09_AI_Systems\02_Tools\Brand_Assets\generate_brand_png.py`

Do not hand-edit PNG files. Change the generator and re-run it.

---

# 6. Known v1 Limits

- Wordmark text in the logo SVG files uses a system font stack (`<text>` elements), so exact letterforms vary slightly by platform. Converting wordmarks to fixed vector outlines is a v2 task.
- The OG image wordmark uses an original 5x7 pixel font by design.
